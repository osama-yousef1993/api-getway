What is High Contrast Theme?

High contrast themes use stark color differences (usually black and white) to improve readability for users with visual impairments, low vision, or in bright lighting conditions.

The data-theme="high-contrast" attribute on the HTML element activates these CSS custom properties to provide maximum contrast between text and backgrounds.

Accessibility Benefits

High contrast themes help users with visual impairments, dyslexia, and those using devices in bright sunlight.

Implementation

Simply add data-theme="high-contrast" to your HTML element and define the CSS custom properties.

How It Works in HTML

1. HTML Structure:

<!-- Default theme -->
<html data-theme="light">

<!-- High contrast theme -->
<html data-theme="high-contrast">

<!-- Or apply to any element -->
<div data-theme="high-contrast">
    <p>This content uses high contrast theme</p>
</div>

2. CSS Definition:

[data-theme="high-contrast"] {
    --theme-bg-primary: #000000;
    --theme-bg-secondary: #ffffff;  
    --theme-text-primary: #ffffff;
    --theme-text-secondary: #000000;
    --theme-border-color: #ffffff;
    --theme-focus-ring: rgba(255, 255, 0, 0.8);
}

3. JavaScript Control:

// Switch to high contrast
document.documentElement.setAttribute('data-theme', 'high-contrast');

// Switch back to light theme  
document.documentElement.setAttribute('data-theme', 'light');
